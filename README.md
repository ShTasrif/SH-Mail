# SH-Mail
A Temporary Email Generator Tool

## Install on Termux :
```
pkg update -y && pkg upgrade -y
pkg install python -y
pkg install git -y
git clone https://github.com/ShTasrif/SH-Mail
```

# Run Termux:
```
cd SH-Mail
python mail.py
```
